## Logout Button Unresponsive in Safari

**Description**  
The logout button fails to trigger any action when clicked in the Safari browser, preventing user logout.

**Steps to Reproduce**  
1. Open Safari browser on any macOS/iOS device  
2. Log in to the application  
3. Navigate to any page containing the logout button  
4. Click the logout button  

**Expected vs Actual Behavior**  
- **Expected**: Clicking the button logs the user out and redirects to the login screen.  
- **Actual**: No response occurs when clicking the button; the user remains logged in.  

**Environment**  
Authentication Component (Safari browser, macOS/iOS devices)  

**Severity**  
Critical (blocks account security/logout functionality)
