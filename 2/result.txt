## Title: Logout Button Unresponsive in Safari Browser  

**Description**: The logout button fails to respond when clicked in the Safari browser, preventing users from ending their session.  

**Steps to Reproduce**:  
1. Log in to the application using valid credentials.  
2. Navigate to any page containing the logout button.  
3. Click the logout button.  

**Expected vs Actual Behavior**:  
- **Expected**: The user session terminates, redirecting to the login page.  
- **Actual**: The logout button is unresponsive, and the session remains active.  

**Environment**: [Authentication] (Safari/macOS; specific version Unknown)  

**Severity**: [Major] (Impairs core user flow but doesn't block all functionality)
